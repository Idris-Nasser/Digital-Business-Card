# 💳 Digital Business Card

A modern and minimalist personal vCard designed to showcase professional identity, skills, and contact information efficiently.

---

## ✨ Key Features

* 🟢 **Availability Indicator:** A smart status dot to instantly display your current availability for work.
* ⚡ **Interactive Hover Effects:** Smooth animations on skill badges and contact buttons to enhance user experience.
* 📱 **Fully Responsive Design:** Built with CSS Media Queries to ensure the card looks great on all devices, from mobile phones to desktops.
* 🎨 **Attractive Dark Theme:** Features a professional dark gradient background and clean typography (Cairo font) for a sleek, eye-catching look.

---

## 🛠️ Technologies Used

* **HTML5:** Structured for semantic organization of profile details, skills, and links.
* **CSS3:** Utilized `Flexbox` and `Grid` for layout management, along with advanced features like `transitions` and `linear-gradients`.
